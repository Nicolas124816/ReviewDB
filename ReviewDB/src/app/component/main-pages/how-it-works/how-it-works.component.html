How It Works

Utilizing Elasticsearch and TMDB, we're engineering a search tool that finds movies by matching user input to movie reviews and comments rather than name or description. This allows users to connect with movies in a way that reflects other movie-watchers experiences rather than an idealized synopsis. With our tool we hope to help users find new movies they truly connect with.

The Process

We enter your inquery into our Elasticsearch which accesses TMDB's data, which is updated daily 8:00 AM UTC. The results are analyzed and results with over an 80% match are presented to you. To connect the back end (the search and analysis) to the front end (this website) we are using the tool Django. Our front end is run in Angular.

Tools used in this project

  2. Docker
